### Additional Guides and Learning Resources

- **Open e-learning course "HPC Basics"** *(in Latvian)*  
  [https://hpc-pamati.learning.lv/](https://hpc-pamati.learning.lv/)  
  An online course covering the fundamentals of High Performance Computing.

- **RUDENS HPC Cluster User Guide**  
  [https://hpc-guide.rtu.lv](https://hpc-guide.rtu.lv)  
  Comprehensive documentation for accessing and using the RUDENS HPC system.

- **HPC Application Examples with Source Code**  
  [https://gitlab.com/eurocc-latvia](https://gitlab.com/eurocc-latvia)  
  A collection of reusable code examples demonstrating HPC applications across various scientific domains.
